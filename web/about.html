Drupal?

Make it even easier to figure out if your favorite sites are running your favorite CMS. Just drag and drop the link above to your browser's bookmarks bar and you've got a quick and easy way to check any site.

How does all of this work?

This site uses a PHP script to perform a couple of simple tests on the provided url. It first checks for the existence of a file named misc/drupal.js, failing that it checks the HTTP Expires header returned by the site. If said header is equal to Sun, 19 Nov 1978 05:00:00 GMT we can pretty much assume that it is Drupal.
